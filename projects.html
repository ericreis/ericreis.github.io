CGraph

  • Overview
  • Technologies Used
  • Source Code

Overview

An undervelopment C++ graph library. It makes use of most known data structures from STL and some implementations. It uses templates, so it works with any data structure the user would like for the Graph (AdjacencyVector, AdjacencyMatrix and AdjacencyMatrixPlus). The AdjacencyMatrixPlus is implemented with std::bitset so it's able to store graphs with 500K nodes with a 16Mb memory system.
